YOLO全系列版本总览（含官方与主流衍生版）

以下按**时间线**整理 YOLO 从 2015 年至今的完整版本，含核心改进、发布团队与关键特性，帮助快速定位各版本差异。

---

### 一、原始奠基阶段（Redmon 团队，2015-2018）

|   |   |   |   |   |
|---|---|---|---|---|
|版本|发布时间|核心特点|骨干网络|关键改进|
|**YOLOv1**|2015/2016|**单阶段检测开山之作，端到端回归**|自定义 CNN（类似 GoogLeNet）|**7×7 网格预测，每个网格 2 个框 + 类别概率；45 FPS，VOC mAP 63.4%**|
|**YOLOv2/YOLO9000**|2017|**更快更强，支持 9000 + 类别**|Darknet-19|**引入 Anchor Boxes；BatchNorm；多尺度训练；维度聚类优化框先验**|
|**YOLOv3**|2018|**多尺度检测突破，小目标性能提升**|Darknet-53（残差结构）|**FPN 特征金字塔；3 尺度预测；二元交叉熵支持多标签分类；mAP 达 83.6%**|

### 三、版本关键演进脉络

1. **检测范式**：从 Anchor-based（v2-v8）→ 混合 Anchor-free（v9）→ 纯 Anchor-free（部分衍生版）
    
2. **骨干网络**：自定义 CNN → Darknet 系列 → CSPDarknet → 高效 Rep/ELAN/GELAN
    
3. **特征融合**：单尺度 → FPN → PANet → RepPAN → 自定义高效融合模块
    
4. **训练优化**：从基础技巧 → "Bag of Freebies" → 动态标签分配 → 神经架构搜索
    
5. **任务扩展**：目标检测 → 实例分割 → 姿态估计 → 多模态融合

### 四、版本选择建议

|   |   |   |
|---|---|---|
|应用场景|推荐版本|理由|
|快速原型开发|YOLOv8/YOLO11|开箱即用，Ultralytics 生态完善，文档丰富|
|边缘设备部署|YOLOv10-n/YOLO11-n|轻量级，高帧率，低资源消耗|
|工业级精度需求|YOLOv9/YOLO11-l|最新架构，更高 mAP，适合高精度场景|
|学术研究对比|YOLOv1-YOLOv12 全系列|覆盖完整演进路线，便于方法验证|
